# Humble CLI

## Install:

**On your Mac:**

```
bash <(curl -s https://raw.githubusercontent.com/marcopeg/humble-cli/master/remote-setup.sh)
```

**On your Ubuntu Server:**

```
bash <(wget -qO- https://raw.githubusercontent.com/marcopeg/humble-cli/master/remote-setup.sh)
```

## Update:

```
humble-update
```
