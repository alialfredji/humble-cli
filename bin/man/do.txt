Docker Humble Utilities
=======================

 - fs-export                move a volume to the host file system
 - fs-import                move a host folder to a container volume
 - fs-dump                  exports a volume to a compressed archive
 - fs-seed                  imports a compressed archive to a volume
 - cleanup                  utilities to cleanup after Docker
 - getip                    gives you the current network ip for your machine

To read the man page for a specific command run:

    ```
    humble help do/fs-export
    ```
