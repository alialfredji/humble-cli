FS-DUMP
=======

Dumps a service volume to a host's compressed archive.

    ```
    humble utils fs-dump storage://data/files
    ```

Will create a compressed archive in the local project cwd:

    ```
    data/backup/storage___data---files___{date.time}.tar.gz
    ```

where `data/backup` is defined in the environment variable `$BACKUP_ROOT`.

You can customize the target folder with both relative and absolute paths,
with or without the name of the target compressed archive:

    ```
    humble utils fs-dump storage://data/files foo
    -> data/backup/foo/storage___data---files___{date.time}.tar.gz

    humble utils fs-dump storage://data/files foo/archive.tar.gx
    -> data/backup/foo/archive.tar.gz

    humble utils fs-dump storage://data/files /foo
    -> /foo/storage___data---files___{date.time}.tar.gz

    humble utils fs-dump storage://data/files /foo/archive.tar.gx
    -> /foo/archive.tar.gz
    ```
